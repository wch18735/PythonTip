### How to Debug
#### Buttons
- step over: move one step
- step into: into function
- step out: go function out
- stop: stop process
- run to cursor: move to cursor

### Tip
- Variable + F2: Change variable
- Variable + add watch: add to Watches view
- Watches: also testing view
